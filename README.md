A full stack kubernetes cluster for self hosting home automation, media services, storage with easy setup / teardown scripts and full observability right out of the box. 

Kubernetes-deployable services that make up a home automation system, intended to be used as a playground and test ground to develop on, or to deploy to a self-hosted or cloud provider as you see fit.

All provided with easy setup teardown scripts that are runnable multiple times without spawning errors.

Full ingress support with communication and security front and foremost. All services on a service mesh and ability to map hardware to specific nodes for affinity.
